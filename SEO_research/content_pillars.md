# Контентные опоры: FAQ и раздел «Как строить из газобетона»

## Источники данных
- Топ‑100 запросов по регионам (`*_semantic_master.csv`) → агрегированы в `semantic_summary.md`.
- Запросы «быстрых побед» (`high_freq_low_comp.json`) и словарь `data/seo/region-query-data.ts`.
- Кластеры FAQ и статей (`faq_topics.json`, `construction_topics.json`).

## Общая стратегия
1. **Работаем по кластерам.** Каждая статья/ответ привязывается к кластерам: `цены`, `доставка`, `монтаж`, `утепление`, `калькулятор`, `бренды`.
2. **Используем региональные формулировки.** В текстах обязательно упоминаем географию: «газобетон Самара», «утепление газобетона в Москве», «доставка газобетона СПб».
3. **Сохраняем перелинковку.** Внутренние ссылки: каталог → FAQ → /construction → доставка → калькулятор. Каждая публикация должна вести на коммерческие страницы.
4. **Поддерживаем JSON-LD.** Готовые шаблоны (`Product`, `CollectionPage`, `FAQPage`, `BreadcrumbList`) уже на месте. В новых статьях используем `Article` / `HowTo` (см. блок «AI-индексация» ниже).

## Рекомендации для AI-индексации и структурированных данных
- **Уже реализовано:** `Product`, `Offer`, `FAQPage`, `CollectionPage`, `BreadcrumbList`, `LocalBusiness`. Все JSON-LD генерируются динамически из контента и запросов.
- **Следующие шаги:**
  - Для статей `/construction` использовать `Article` или `HowTo` (если будет пошаговая инструкция). Добавлять `author`, `headline`, `keywords`, `image`.
  - Для чек-листов/таблиц применять `HowToStep` или `HowToDirection`, если описываем монтаж.
  - Для сравнения брендов → `Product` + `AggregateRating` (как только будут реальные отзывы/кейсы).
- **AI-оптимизация текста:** придерживаться «ответ-объяснение-пример-CTA». Встраивать «неявные» вопросы (например, «Можно ли построить дом зимой из газобетона?») прямо в текст, выделять их подзаголовками.
- **Технические требования:** уникальные описания (≥300 слов), таблицы и списки для структурирования, alt-тексты с ключами, поддержка русской морфологии (склонения «газобетон в Самаре», «доставка газобетона по Санкт-Петербургу»).
