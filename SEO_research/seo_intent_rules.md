## Правила классификации SEO-запросов

### 1. Определение интента
- **Брендовый** — если в запросе есть название или аббревиатура производителя (СГЗСБ, Газобетон 63, Новоблок, Aeroc, Ytong, Bonolit, ГРАС, Теплон, LSR, Глеб Грин, Поритеп и т.д.).
- **Коммерческий** — если встречаются триггеры покупки: `купить`, `цена`, `стоимость`, `заказать`, `доставка`, `в наличии`, `склад`, `компания`, `оптом` и т.п., либо это короткий региональный запрос по газобетону.
- **Информационный** — если присутствуют вопросы или слова, связанные с обучением/эксплуатацией: `как`, `чем`, `почему`, `нужно ли`, `инструкция`, `монтаж`, `утепление`, `армирование`, `расчет`, `калькулятор`, `плюсы`, `минусы`, `сравнение`, `толщина`, `вес` и т.д. Если нет явных сигналов, приоритет у коммерческого для общих региональных ключей.

### 2. Привязка к странице/группе
- `/[region]/catalog` — основная категория для запросов о покупке, ценах, наличии.
- `/[region]/delivery` — если упоминаются доставка, логистика, манипулятор.
- `/[region]/calculator` — для запросов с `калькулятор`, `расчет`, `сколько блоков` и т.д.
- `/construction` — все чисто информационные запросы про строительство, монтаж, утепление, сравнения, инструкцию.
- `/[region]/catalog/<brand>` — брендовые запросы со слагом производителя (добирается из кластера).

### 3. Кластеры
- Сначала проверяется бренд: `бренд: <название>`.
- Далее ключевые темы: `цены`, `покупка`, `доставка`, `калькулятор`, `характеристики`, `монтаж`, `утепление`, `конструкции`, `перегородки`, `сравнения`, `проекты зданий`, `инструменты`, `крепёж`, `фундамент`, `эксплуатация` и т.п.
- Если ничего не подошло — `общий спрос`.

### 4. Приоритеты
- Частота (или demand) ≥ 200 → `1`.
- Частота ≥ 60 → `2`.
- Если источник содержит `Webmaster` и частота ≥ 30 → `2`.
- Остальное → `3`.

### 5. Тип хвоста
- До двух слов → `основной`.
- Наличие вопросительных конструкций → `вопрос`.
- Остальные → `длинный`.

### 6. Статус
- Приоритет `1` или `2` → `к проработке`.
- Приоритет `3` → `наблюдать`.

### 7. Контент/UX задачи
- Для каждого кластера предусмотрен шаблон рекомендации:
  - `цены` → актуализировать блок цен и условия покупки.
  - `доставка` → уточнить тарифы, сроки, опции логистики.
  - `калькулятор` → дополнить инструкциями и CTA.
  - `характеристики` → добавить таблицы параметров и пояснения.
  - `монтаж`, `утепление`, `конструкции` → расписать этапы, советы, схемы.
  - `сравнения` → подготовить материалы по сравнению газобетона с альтернативами.
  - `бренд:` → сделать бренд-лендинг с описанием завода и фильтром по товарам.
  - По умолчанию — «Расширить контент и CTA под региональные запросы».
